กำลังตรวจสอบสถานะ...

แต่ละรอบระบบจะ นำเข้าวิชาใหม่จากระบบขาย แล้วดึงคำสั่งซื้อสถานะ "ส่งแผนเสร็จสิ้น" มาให้สิทธิ์อัตโนมัติตามความถี่ที่ตั้ง (กันซ้ำให้ทั้งวิชาและสิทธิ์)

📖 คู่มือการใช้งาน

สำหรับผู้ดูแล (เจ้าของระบบ)

  1. จัดการวิชา — เพิ่มวิชาที่เปิดขาย (เช่น คณิตศาสตร์ ป.1)
  2. จัดการสื่อ — เลือกวิชา แล้วเพิ่มลิงก์สื่อแต่ละแผน (URL จาก Apps Script) หรือ นำเข้าทีเดียวจากไฟล์ .xlsx/.csv
  3. จัดการสิทธิ์ — เลือกวิชา ใส่ Gmail ของลูกค้าที่ซื้อ (ใส่หลายอีเมลพร้อมกันได้) กำหนดวันหมดอายุได้
  4. ซิงค์จากคำสั่งซื้อ — ถ้าตั้งค่า CONFIG เชื่อมชีตคำสั่งซื้อแล้ว กดเพื่อดึงสิทธิ์อัตโนมัติ (กดพรีวิวก่อนได้)
  5. สำรองข้อมูล สม่ำเสมอ เผื่อกู้คืนภายหลัง

สำหรับลูกค้า

เข้าเว็บ → กรอก Gmail ที่ใช้สั่งซื้อ → เห็นเฉพาะวิชาที่ตนมีสิทธิ์ → กด "เปิดสื่อ" ได้ทุกอุปกรณ์

การเชื่อมชีตคำสั่งซื้อ (CONFIG ใน Code.gs)

  1. ใส่ ORDER_SHEET_ID และชื่อแท็บ
  2. แก้ ORDER_COL ให้ตรงชื่อคอลัมน์จริง (อีเมล/สินค้า/สถานะ)
  3. จับคู่ชื่อสินค้า → รหัสวิชา ใน PRODUCT_TO_SUBJECT

วิธี Deploy เป็น Web App

  1. สร้าง Google Sheet ใหม่ → เมนู ส่วนขยาย (Extensions) → Apps Script
  2. วางโค้ด Code.gs และสร้างไฟล์ HTML ชื่อ index วางโค้ด index.html
  3. กด การทำให้ใช้งานได้ (Deploy) → การทำให้ใช้งานใหม่ → เลือกชนิด: เว็บแอป
  4. ดำเนินการในชื่อ: ฉัน (Me) · ผู้มีสิทธิ์เข้าถึง: ทุกคน (Anyone)
  5. คัดลอก Web App URL ไปแชร์ให้ลูกค้า
  6. อัปเดตโค้ดทุกครั้ง: การทำให้ใช้งานได้ → จัดการการทำให้ใช้งานได้ → ✏️ แก้ไข → เวอร์ชัน: ใหม่ → ใช้งาน
⚠️ หมายเหตุความปลอดภัย: ระบบนี้คุม "การมองเห็น" ลิงก์ หากลูกค้านำลิงก์สื่อ (URL สาธารณะ) ไปแชร์ต่อ ผู้อื่นยังเปิดได้ หากต้องการล็อกระดับลิงก์จริง ต้องเพิ่มการตรวจสิทธิ์ในไฟล์สื่อด้วย
